# SpaceWhisper – Astronomical Web Portal

SpaceWhisper is an interactive web platform dedicated to astronomy, designed to make space exploration more accessible, engaging, and educational for everyone; from curious beginners to passionate enthusiasts.

---

## Project Overview

SpaceWhisper unifies intuitive interface, education, exploration, technology insights, and community into a single astronomy platform.

---

## Features

### Implemented Modules

-  **Big Questions**
  - Simplified explanations of complex astrophysics concepts

-  **Technology in Astronomy**
  - Presentation of modern astronomical technologies

---

###  Partially Implemented

-  **Explore the Cosmos**
  - UI ready (API integration pending)

-  **Contribute to Astronomy**
  - Redirect interface to citizen science projects

---

###  Planned Features

-  News from the Universe
-  Ask the Cosmos (AI chatbot)
-  Learn & Test Yourself (quiz system)
-  Community platform
